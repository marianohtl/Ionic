ionic via node

Abra o prompt de comando de preferencia no modo administrador e execute o comando abaixo: npm install -g ionic cordova

Criando o projeto

Para criar o projeto é bem simples, vamos utilizar o comando abaixo:
ionic start NomeSeuApp tabs --type=angular

Logo em seguida digite o seguinte comando no prompt
cd NomeSeuApp
code.
Code irá abrir o Visual Studio Code na pasta do projeto

Um projeto será criado com uma estrutura semelhante a está

Rodando o projeto

Para seu projeto abrir no navegador, digite no prompt de comando:
ionic serve
